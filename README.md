# Phishing Website Detection using Machine Learning

## Overview
This project detects phishing websites using machine learning techniques based on URL features.

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest (Final Model)

## Accuracy
- Cross-validation accuracy: ~91%

## Features
- URL-based feature extraction
- Real-time prediction using Flask
- Model evaluation using ROC, Confusion Matrix, Precision-Recall

## Technologies
- Python
- Scikit-learn
- Flask
- Pandas, NumPy

## How to Run
1. Install dependencies:
pip install -r requirements.txt
2. Run app:
python app.py
3. Future Improvements
- Deep learning models
